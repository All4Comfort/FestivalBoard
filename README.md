# 프로젝트 소개
<div align="center">
  <h3>Fullstival = Full + Festival</h3>
  <p>Spring Boot를 기반으로 축제 정보를 찾아보고 방문자들 간에 리뷰를 공유할 수 있는 게시판형 웹사이트를 제작했습니다.</p>
